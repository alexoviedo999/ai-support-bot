🤖 AI Support Widget Demo

Click the chat bubble in the bottom-right corner to start a conversation!

This is a Phase 1 demo - the bot currently returns a placeholder response. In Phase 2, it will answer questions from uploaded documents using RAG.

Embed Code

Add this to any website:

<script src="https://ai-support-widget.vercel.app/ai-support-widget.umd.js"></script>
<script>
  AISupportWidget.init({
    tenantId: 'your-tenant-id',
    botName: 'Support Bot',
    primaryColor: '#6366f1'
  });
</script>
